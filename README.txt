# PSX-script-.lua

El archivo .lua contiene los cheats del juego

Cada archivo .lua es especifico de cada juego, se debe configurar por ID (SLUSXXXXX-SCUSXXXXX-SLESXXXXX-SLPMXXXXX-SCESXXXXX) y por region (NTSC,PAL,NTSC-J).
Los codigos son de cualquier programa conocido de implementacion de trucos Gameshark, Action Replay, Xploder, etc.
tratare de subir mas archivos conforme a mis tiempos les pido paciencia.
Por el momento cuento con 2 archivos funcionales de los juegos Peter Pan Return to the Neverland version PAL y para Castlevania Symphony of the Night version NTSC

Cabe mencionar que una vez se crea el pkg con el programa psx2ps4 v.1.05 cada vez que juegues la partida empezara con los trucos activados, no hay un metodo todavia para activar o desactivar esa funcion.
